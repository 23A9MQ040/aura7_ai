# 🌌 AURA — Adaptive Universal Reasoning Assistant

AURA is a futuristic personal AI operating system powered by a multi-agent architecture. It understands humans, adapts continuously, and takes intelligent actions while keeping user data secure.

## 🚀 Vision
"An AI ecosystem that understands humans, adapts continuously, and takes intelligent actions while keeping user data secure."

## 🌟 Key Features
- **Multi-Agent Intelligence**: Seven specialized agents (Learning, Career, Health, Finance, Impact, Memory, Privacy).
- **Privacy Guardian**: Complete data sovereignty with end-to-end encryption and audit trails.
- **Autonomous Reasoning**: Breaks complex goals into actionable, verifiable steps.
- **Persistent Memory**: Long-term AI memory that evolves with you.
- **Social Impact**: "Agents for Good" assisting farmers, students, and accessible tech.

## 🏗️ Tech Stack
- **Frontend**: Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion, Three.js
- **Backend**: Java 20, Spring Boot (Microservices)
- **Database**: PostgreSQL, Redis, ChromaDB (Vector)
- **DevOps**: Docker, Kubernetes, GitHub Actions

## 💻 Quick Start (Frontend)
1. \`cd frontend\`
2. \`npm install\`
3. \`npm run dev\`
4. Open [http://localhost:3000](http://localhost:3000)

See \`DEPLOYMENT.md\` for full stack Docker setup.
